Windows Azure Deployment Project properties

This window allows users to edit the project properties of Windows Azure Deployment Project.

You can see the list of all roles present in the project.

Click on Add \ Edit \ Remove buttons for specified operations on role.

While adding the role, role's name must be unique.

Specify VM Size using the combo box. Set this value to change the size of the VM allotted to this role. The default value is Small. Other possible values are ExtraSmall, Medium, Large, and ExtraLarge.

Number of Instances element specifies the number of instances of this role to deploy as a part of the service deployment.
